    My MkDocs Tutorial¶

    Welcome to the MkDocs tutorial for beginners! Here, you'll learn how to create content in MkDocs using simple Markdown syntax.

    1. Heading and Subheading¶

    You can create headings using hash symbols (#).

    1.1. Subheading¶

    Subheadings are created by adding more hash symbols (## for subheadings, ### for smaller subheadings, and so on).

    For example:

    # Main Heading
## Subheading Level 1
### Subheading Level 2

    2. Writing Text and Paragraphs¶

    You can add simple text to your page. Simply type your content, and Markdown will automatically format it as paragraphs. For example:

    This is a simple paragraph of text in MkDocs. MkDocs will automatically render paragraphs and organize them properly.
    Just separate paragraph by leaving a blank line between them

    Example Paragraphs:¶

    Here’s how you might write content with multiple paragraphs:

    This is the first paragraph.

This is the second paragraph. It's also important to remember that MkDocs uses Markdown syntax to render text.

    3. Emphasizing Text¶

    You can emphasize text by using italic or bold syntax.

    Italic Text is wrapped in one asterisk (). Bold Text is wrapped in two asterisks (*). Bold and Italic Text uses three asterisks.

    This is *italic* text and this is **bold** text.

    4. Links¶

    To create a hyperlink, use the following syntax:

    [Text to Display](https://www.example.com)

    For Example : Visit Google for more information.

    and this above code is 

    Visit [Google](https://www.google.com) for more information.

    To add an image, use the syntax:

    ![Image Alt Text](path/to/image.jpg)

    6. Adding Images from Your Computer¶

    To add an image from your computer, place the image file in a folder within your MkDocs project (e.g., docs/images/) and reference it in your Markdown file.

    Steps: Place Image in a Folder: Place your image file (e.g., my_image.jpg) in a folder like docs/images/.

    Reference Image in Markdown: Use the following Markdown syntax to add the image:

    ![Image Alt Text](images/my_image.jpg)
    This assumes the my_image.jpg file is located in the docs/images/ directory.

    Example :

    # Adding Images from Your Computer

Here is an image:

![Example Image](images/my_image.jpg)

    7. Embedding a YouTube Video¶

    You can add a YouTube video using an embedded link. To do this, use the Markdown syntax for linking, or you can embed the video using HTML directly.

    Steps: Find the YouTube Video URL: Go to the YouTube video you want to embed, and copy its URL (e.g., https://www.youtube.com/watch?v=dQw4w9WgXcQ).

    Add the YouTube Video Link in Markdown:

    Option 1: Direct Link (Markdown)

    [Watch this video on YouTube](https://www.youtube.com/watch?v=dQw4w9WgXcQ)

    Option 2: Embed the Video (HTML)

    Use an HTML iframe to embed the video directly into the page.

    <iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

    Example: 

    # Embedding a YouTube Video

You can watch this amazing video directly below:

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

    8. Linking to Documents (PDF, Word, PowerPoint, etc.)¶

    To link to external documents like PDF, Word, or PowerPoint files, follow these steps:

    Steps: Place the Document in Your Project: Just like with images, you need to put the document (e.g., document.pdf, presentation.pptx, file.docx) into your docs folder or a subfolder (e.g., docs/files/).

    Reference the Document in Markdown: Use the following Markdown syntax to create a link to the document.

    Example for PDF, Word, or PowerPoint:

    # Link to Documents

Here are some documents you can download:

- [Download PDF Document](files/document.pdf)
- [Download PowerPoint Presentation](files/presentation.pptx)
- [Download Word File](files/file.docx)

    And if you want the pdf to be open in new tab then you can write like following

    <iframe src="files/document.pdf" width="100%" height="600px"></iframe>

    9. How to Add a PowerPoint (PPT) File¶

    MkDocs doesn't render PowerPoint presentations directly, but you can still add a link to download a PowerPoint file, or if you want to display it in a viewable format, you can convert the PowerPoint to PDF or use an embedded service like Google Slides.

    Steps: Upload the PowerPoint File to Your Project: Place the .pptx file in your docs folder or subfolder, e.g., docs/files/.

    Add a Download Link in Markdown:

    [Download PowerPoint Presentation](files/presentation.pptx)

    10. Steps to Add a Video from a Computer (Uploaded Video)¶

    # Uploading and Playing a Video

To upload and play a video from your computer in MkDocs, follow these steps:

1. Place the video in the `docs/videos/` folder.
2. Embed the video in your Markdown file using the HTML5 `<video>` tag.

## Example Video

Here is an example of a video uploaded to the project:

<video width="600" controls>
  <source src="videos/my_video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

    Summary of Markdown Syntax:¶

    • # = Main Heading
    • ## = Subheading
    • Text = Paragraphs (separate with blank lines)
    • - or * = Unordered List
    • 1. = Ordered List
    • *italic* or **bold** = Emphasis
    • [Link](url) = Hyperlink
    • ![Alt Text](url) = Image
